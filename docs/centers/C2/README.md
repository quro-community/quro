# C2: Leaf-dominated Fanout — Utility Layer

> Center C2 | Archetype: fanout | 624 symbols
> Coupled with: C0

## Overview

C2 is a leaf-dominated fanout center in the utility layer of the Quro codebase. It provides traversal mode selection, flow observer utilities, and dictionary conversion helpers. As a fanout archetype, C2 expands outward from leaf entry points, feeding utility functionality into the C0 orchestration hub.

## Entry Points

- `sym::select_traversal_mode` — Traversal mode selection
- `sym::FlowObserver::flow_observer::53` — Flow observer utility
- `sym::to_dict::flow_observer::37` — Dictionary conversion helper
